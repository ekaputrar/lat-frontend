# Tugas Latihan Fullstack: React + Express + PostgreSQL

Proyek ini merupakan latihan fullstack sederhana yang terdiri dari frontend menggunakan React dan backend menggunakan Express.js dengan database PostgreSQL. Aplikasi ini memungkinkan pengguna untuk melakukan registrasi, login, dan mengelola data item pribadi.
